# Ограничение категорий для провайдер-админа

## Статус: ⏳ Не проверено

---

## Вход в админку как провайдер-админ

- [ ] Войти в Django Admin как провайдер-админ (не суперпользователь)
- [ ] Перейти в раздел "Providers"
- [ ] Открыть редактирование своего Provider

---

## Проверка доступных категорий

- [ ] Проверить поле `available_category_levels` в форме редактирования
- [ ] Проверить, что доступны только категории, выбранные в заявке
- [ ] Проверить, что другие категории не отображаются в списке
- [ ] Проверить, что отображается подсказка о том, что можно выбрать только категории из заявки

---

## Попытка выбрать другие категории

- [ ] Попытаться выбрать категорию, которая не была в заявке
- [ ] Проверить, что такая категория недоступна в списке
- [ ] Проверить, что можно выбрать только из одобренных категорий

---

## Проверка для суперпользователя

- [ ] Войти в Django Admin как суперпользователь
- [ ] Открыть редактирование Provider
- [ ] Проверить, что суперпользователь видит все категории (без ограничений)
