# Проверка для неавторизованного пользователя

## Статус: ⏳ Не проверено

---

## Кнопка "Поставщикам услуг"

- [ ] Открыть сайт без авторизации
- [ ] Найти кнопку "Поставщикам услуг" в Header
- [ ] Проверить, что кнопка активна (не disabled)
- [ ] Нажать на кнопку "Поставщикам услуг"

---

## Модальное окно

- [ ] Проверить, что появляется модальное окно
- [ ] Проверить текст сообщения: "Сначала авторизуйтесь в системе" (или перевод)
- [ ] Проверить переводы сообщения на всех языках:
  - [ ] EN: "Please log in first"
  - [ ] RU: "Сначала авторизуйтесь в системе"
  - [ ] DE: "Bitte melden Sie sich zuerst an"
  - [ ] ME: "Molimo se prvo prijavite"

---

## Закрытие модального окна

- [ ] Проверить, что можно закрыть модальное окно
- [ ] Проверить, что после закрытия окна не происходит редирект
