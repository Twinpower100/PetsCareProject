# Тестовые скрипты для функциональности добавления провайдеров

## Структура тестовых скриптов

Эта папка содержит тестовые скрипты для проверки функциональности добавления провайдеров в систему.

### Структура папок:

1. **01_Форма_заявки_провайдера_Frontend/** - Тестирование фронтенд части формы заявки
2. **02_API_и_Backend/** - Тестирование API и бэкенд функциональности
3. **03_Одобрение_заявки/** - Тестирование процесса одобрения заявок
4. **04_Ограничения_в_админке/** - Тестирование ограничений доступа в админке
5. **05_Проверка_авторизации/** - Тестирование проверки авторизации
6. **06_Переводы_и_локализация/** - Тестирование переводов и локализации
7. **07_Дополнительные_проверки/** - Дополнительные проверки

## Как использовать

1. Откройте соответствующий файл со скриптом тестирования
2. Выполните каждый пункт чеклиста
3. Отмечайте выполненные пункты галочкой `[x]`
4. Заполняйте раздел "Заметки" с результатами тестирования
5. Указывайте дату тестирования и имя тестировщика

## Статусы тестирования

- ⏳ Не проверено - Тест еще не выполнялся
- ✅ Пройдено - Все пункты теста пройдены успешно
- ❌ Провалено - Обнаружены проблемы
- ⚠️ Частично - Часть пунктов пройдена, есть проблемы

## Общие замечания

- Все тесты должны выполняться последовательно
- При обнаружении проблем, опишите их в разделе "Заметки"
- После завершения всех тестов, обновите статус в заголовке файла
