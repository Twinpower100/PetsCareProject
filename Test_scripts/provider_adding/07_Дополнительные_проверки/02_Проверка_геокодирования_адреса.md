# Проверка геокодирования адреса

## Статус: ⏳ Не проверено

---

## Геокодирование при одобрении

- [ ] Одобрить заявку с адресом в админке
- [ ] Проверить, что адрес геокодируется через Google Maps API
- [ ] Проверить, что создается `StructuredAddress` для Provider
- [ ] Проверить, что координаты (latitude, longitude) установлены корректно
- [ ] Проверить, что `Provider.point` установлен корректно

---

## Обработка ошибок геокодирования

- [ ] Попытаться одобрить заявку с некорректным адресом
- [ ] Проверить, что ошибка геокодирования обрабатывается корректно
- [ ] Проверить, что Provider все равно создается (если адрес не критичен)
- [ ] Проверить логи на наличие ошибок геокодирования
