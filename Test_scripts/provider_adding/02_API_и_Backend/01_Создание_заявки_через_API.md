## Валидация в API

- [ ] Отправить заявку без категорий → должна вернуться ошибка валидации
- [ ] Отправить заявку с категорией уровня не 0 → должна вернуться ошибка
- [ ] Отправить заявку с несуществующим ID категории → должна вернуться ошибка
- [ ] Отправить заявку без обязательных полей → должны вернуться ошибки валидации
